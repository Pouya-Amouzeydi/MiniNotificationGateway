# Mini Notification Gateway

Final project for Design Patterns in C# course.

## Project Goal

This project implements a simple notification gateway that can create and send OTP messages using multiple SMS providers.

If the first provider fails, the system automatically switches to the next provider.

## Main Requirements

- Create OTP messages
- Send messages using SMS providers
- Support ProviderA and ProviderB
- Provider failover
- Message status management
- Event logging
- Dependency Injection
- SOLID principles
- Clean code
- Design Patterns usage

## Technologies

- C#
- .NET 10
- Console Application
- Microsoft Dependency Injection
